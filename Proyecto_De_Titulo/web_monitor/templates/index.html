Plataforma de adquisición de parámetros de voz y variables ambientales

Bienvenido a la plataforma de monitoreo y análisis de datos. Aquí podrás gestionar instituciones, profesores, dispositivos IoT y visualizar estadísticas de manera eficiente.

Utiliza el botón de abajo para desplazarte rápidamente a la guía de uso.

Cómo usar la plataforma

Instituciones

En la página de instituciones, puedes realizar las siguientes acciones:

  • Agregar Institución: Haz clic en el botón "Agregar Institución" para registrar una nueva institución en el sistema.
  • Exportar a Excel: Utiliza el botón "Exportar a Excel" para descargar un archivo con la lista de instituciones.
  • Buscar Instituciones: Usa el buscador para encontrar instituciones por nombre o RUT.
  • Ver Detalles: Haz clic en una institución para ver información detallada.
  • Editar Institución: Modifica los datos de una institución existente.
  • Eliminar Institución: Elimina una institución del sistema.
  • Gestionar Aulas: Accede a la gestión de aulas asociadas a una institución.
Profesores:

En la página de profesores, puedes realizar las siguientes acciones:

  • Agregar Profesor: Haz clic en el botón "Agregar Profesor" para registrar un nuevo profesor en el sistema.
  • Exportar a Excel: Utiliza el botón "Exportar a Excel" para descargar un archivo con la lista de profesores y sus horarios.
  • Buscar Profesores: Usa el buscador para encontrar profesores por nombre o ID.
  • Ver Detalles: Haz clic en un profesor para ver información detallada, incluyendo sus horarios.
  • Editar Profesor: Modifica los datos de un profesor existente.
  • Eliminar Profesor: Elimina un profesor del sistema.
  • Gestionar Horarios: Accede a la gestión de horarios asociados a un profesor.
Dispositivos IoT:

En la página de Dispositivos IoT, puedes realizar las siguientes acciones:

  • Agregar Dispositivo IoT: Haz clic en el botón "Agregar Dispositivo IoT" para registrar un nuevo dispositivo en el sistema.
  • Exportar a Excel: Utiliza el botón "Exportar a Excel" para descargar un archivo con la lista de dispositivos registrados.
  • Buscar Dispositivos: Usa el buscador para encontrar dispositivos por nombre o ID.
  • Ver Detalles: Haz clic en un dispositivo para ver información detallada, incluyendo su estado y configuración.
  • Editar Dispositivo: Modifica los datos de un dispositivo existente.
  • Eliminar Dispositivo: Elimina un dispositivo del sistema.
  • Asignar Dispositivo: Asocia un dispositivo IoT a una institución o aula específica.
Visualización de datos:

En la página de estadísticas, puedes realizar las siguientes acciones:

  • Seleccionar Institución: Elige una institución para filtrar los datos relacionados.
  • Seleccionar Profesor: Filtra los datos por profesor asociado a la institución seleccionada.
  • Definir Rango de Fechas: Establece una fecha y hora de inicio y fin para consultar los datos.
  • Consultar: Haz clic en el botón "Consultar" para visualizar los datos en gráficos interactivos.
  • Gráfico Lineal: Si las fechas son iguales, se mostrará un gráfico lineal con los datos correspondientes.
  • Diagrama de Cajas: Si las fechas son diferentes, se mostrará un diagrama de cajas para analizar la distribución de los datos.
  • Exportar Datos: Descarga los datos en formato Excel para un análisis más detallado.
